== Description ==
This plugin shows the status of your Minecraft server using a shortcode and a widget.

* Shortcode - For the shortcode please use [mcserver] or [mcserverplayer]
* Widget - Set your sidebar widget under dashboard and then design

== Installation ==
1. Download the plugin and unzip it in the "/wp-content/plugins/" directory
2. Enable the plugin in the adminpanel under "Plugins"
3. Go to "MCPing" and write your server adress and server port
4. Use the widget and the shortcode (Widgets can you activate under "Appearance -> Widgets")
